Hello World with Backbone framework

It is amazing how I am already half way through the Makersquare bootcamp. I remembered the first day I walked into ROC, I was skeptical about how much I would learn from the program. The learning process took off on the first day that I got there. The program covers some of the most important concepts in computer science such as advanced algorithms and data structures. Then we dove deep into javascript object oriented programming and tackled some difficult topics such as backbone and D3. We made our way to the backend where we learned server sided technologies such as node.js and the different types of databases. When we finished our curriculum working with the angular framework, I personally thought that D3 and backbone.js were more challenging than the other topics from the program. Although, backbone is lightweight MVC framework compare to Ember or Angular but I find myself not understanding the ‘big picture’ on nested model/collection, and how all the components work together. The way events are bound and rendered in the various views actually seems like a lot of entanglements. Below here is an example hello world code base on backbone: 
                     //Create model and class where the data goes to

                        var HelloWorld = Backbone.Model.extend({

                          defaults: function() {
                            return {
                              name: “”
                            }
                          }
                        });


                    //Create instance of the model

                      var helloWorld = new Greeting({
                        name: “Hello World!”
                    });


                    //Create view class to render in HTML

                    var SayHello = Backbone.Model.extend({
                      el: $(‘body’),

                      initialize: function(){
                        _.bindAll(this, ‘render’);
                        this.render();
                    },

                        render: function(){
                          $(this.el).append(“Hello world”);
                        }
                    })

                    //Create instantiation view of greeting

                        var sayHello = new SayHello({
                            model: helloWorld
                        });
I also have to handed to D3. At first, they look beautiful, friendly, and sexy, but i can’t seem to understand how you can use to manipulate documents based on data. I am glad we were able to learn them. It encourages me to read and watch more videos about backbone and D3 on my own time. It seems the best way to learn about backbone js and D3 is through practices. I am not certain to say i understand them completely but i definitely know more about them.
